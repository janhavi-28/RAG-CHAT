# RAG Research Agent

This is a Research Agent application built with Streamlit that leverages Retrieval-Augmented Generation (RAG) using Google Gemini LLM, FAISS vector search, and web scraping to answer user queries based on uploaded PDFs and web search results.

## Features

- Upload a PDF document and ask questions about its content.
- Search the web for relevant information using SerpAPI or fallback scraping.
- Use embeddings and FAISS for efficient similarity search on document chunks.
- Summarize retrieved information with Google Gemini LLM.
- Display answers with sources in a Markdown formatted report.

## Requirements

- Python 3.8+
- Streamlit
- PyPDF2
- google-generativeai
- requests
- beautifulsoup4
- faiss-cpu
- sentence-transformers
- numpy
- python-dotenv

## Setup

1. Clone the repository or download the source code.
2. Create and activate a virtual environment:
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows: venv\Scripts\activate
   ```
3. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
4. Create a `.env` file in the project root and add your API keys:
   ```
   GEMINI_API_KEY=your_gemini_api_key
   SERPAPI_API_KEY=your_serpapi_api_key
   ```
5. Run the app:
   ```bash
   streamlit run rag_research_agent.py
   ```

## Usage

- Upload a PDF file (optional).
- Enter a question in the input box.
- Click "Search" to get a detailed answer with sources from the PDF and/or web.

## Notes

- If no PDF is uploaded, the agent will search the web only.
- The app uses a fallback scraping method if SerpAPI is not available.
- The answer is generated by combining retrieved chunks and summarizing with Gemini LLM.
